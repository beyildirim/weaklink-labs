# Tiers 2-8: Coming Soon

The following tiers are under active development. Tier 0 and Tier 1 are available now.

---

## Tier 2 -- Build Pipeline Security

Attacks targeting CI/CD systems: poisoned build scripts, GitHub Actions exploitation, secrets exfiltration from build environments, and build provenance verification.

## Tier 3 -- Container Supply Chain

Registry compromise, base image poisoning, Dockerfile injection, SBOM generation and verification, and image signing with Cosign.

## Tier 4 -- Code Signing & Provenance

Sigstore ecosystem (Cosign, Fulcio, Rekor), in-toto attestations, SLSA framework compliance, and end-to-end provenance chains.

## Tier 5 -- Runtime Security

Detecting supply chain attacks at runtime: behavioral analysis, syscall monitoring, network policy enforcement, and eBPF-based detection.

## Tier 6 -- Policy Enforcement

Open Policy Agent (OPA), Kyverno, admission controllers, and automated policy gates for the entire supply chain.

## Tier 7 -- Incident Response

Supply chain breach detection, forensic analysis of compromised packages, coordinated disclosure, and recovery procedures.

## Tier 8 -- Red Team Exercises

Full end-to-end attack scenarios combining techniques from all previous tiers. Multi-stage attacks against realistic infrastructure.
